How to compile the source code:
1. Open the SPA project by double-clicking the EmptyGeneralTesting.sln file using Visual Studio 2015.
2. Compile the program in Release mode and run using AutoTester. After compiling in Release mode, AutoTester.exe will be located in the "Release" folder in the solution directory. The command to run AutoTester has the following syntax:
AutoTester.exe <SIMPLE Source> <Query File> <Output File>
